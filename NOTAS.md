## Main Concept

The logo is based on a geometric fusion between:

- letter "J"
- letter "D"

# ISOTYPE DESCRIPTION

## Structure

Geometric angular monogram.

Composition:

- left side forms a stylized "J"
- right side forms a futuristic "D"
- negative space used intentionally
- balanced symmetry
- clean vector geometry

The icon must work:

- as favicon
- as app icon
- on GitHub
- on social avatars
- on dark and light backgrounds
